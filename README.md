# Aleks' Linux Dotfiles and Scripts

In this repository you will find configuration files for many common Linux programs I use in my workflows. Please note that this in no way represents a complete, out-of-the box configuration. You may need other programs/configs/dependencies not found here to configure a complete system. Nevertheless, I hope you find the info here useful. Feel free to copy/use/distribute and of the code here for your personal use.

This repository is managed using the "git bare" method. Find out more [here](https://www.atlassian.com/git/tutorials/dotfiles).
